

nat network : vm 마다 사설 ip 를 부여 받음. vm 끼리 통신 가능, vm -> 외부 네트워크로도 통신 가능

출발 포트(같은 호스트에서도 구글, 네이버등 다른 요청이 나갈 수 있기 때문에 식별하기 위해 존재, OS 가 요청마다 임위로 하나 뽑아서 사용함.)

구글에 vm 이 요청보낼떄
vm1 의 사설 ip : 10.0.2.4:12342

-> host 의 사설 ip : 192.168.0.2:43213 (virtualbox 내부 변환 테이블 에저장됨)
-> 공유기의 공인 ip : 14.50.190.128:81231 (공유기 내부 변환 테이블에 저장됨)
-> 구글 ip : 142.xxx.xxx.xxx:443 (여기 포트는 목적지 포트다. 이건 바뀌지 않음.)
