서블릿은 url 에 매핑되어 해당 url 로 요청이 들어왔을때 서블릿 컨테이너에 의해 service() 가 호출되어 적절한 비즈니스 로직을 실행하게 하는 웹 컴포넌트 이다. mvc 에서 컨트롤러와 비슷한 역활을 한다.


Servlet Container 
서블릿 컨테이너는 http 요청에 맞는 서블릿을 실행시키는 존재이다. 해당 http 요청과 매핑되어 있는 서블릿을 찾아 service() 를 실행시킨ㄷ. service() 가 실행되면 그 안에서 http 메서드를 보고 doPost(), doGet() 등의 메서드를 실행시킨다.
